# ドラフト指名結果メーカー 修正版 v2

## 修正内容

- 5〜6チーム時に各チーム2名しか表示されない問題を修正
- 選手リストを `flex` から `grid-template-rows: repeat(6, 1fr)` に変更
- カード内の文字サイズ・余白を縮小
- 大会名を少し小さく調整
- ノイズを強めに再生成
- ノイズ画像の重ね方を調整
- GitHub Pages向けの相対パスを維持

## アップロード方法

ZIPを解凍し、以下がリポジトリ直下にある状態でアップロードしてください。

```text
index.html
style.css
script.js
README.md
sample-data.json
assets/
```

`index.html` がリポジトリ直下にある状態が安全です。


## v3 変更内容

- 毛筆感を強めるため Google Fonts の `Yuji Syuku` を追加
- 大会名・チーム名を毛筆寄りフォントへ変更
- 文字の黒縁・発光・歪みを強化
- 参考画像に近い荒々しい筆文字感へ調整

注意: フォントはGoogle Fontsから読み込むため、オフライン環境では代替フォントになります。


## 玉ねぎ楷書「激」無料版対応

フォント著作権の関係でフォント本体は同梱していません。

assets/fonts/ に以下の名前で配置してください。

- onion-geki.woff2（推奨）
または
- onion-geki.ttf

配置すると、大会名・チーム名に自動適用されます。
